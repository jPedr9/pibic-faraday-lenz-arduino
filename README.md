# Experimento de Indução Eletromagnética com Arduino

Projeto PIBIC desenvolvido na UESPI para estudo das Leis de Faraday e Lenz utilizando Arduino e aquisição de dados em tempo real.

## Objetivos

- Demonstrar indução eletromagnética
- Visualizar f.e.m. induzida
- Aplicar derivada discreta em sinais experimentais
- Desenvolver abordagem didática para ensino de eletromagnetismo

## Equação principal

E = -N(dΦ/dt)

## Tecnologias

- Arduino Uno
- Python
- PySerial
- Matplotlib

## Resultados

O sistema detecta variações do fluxo magnético geradas pelo movimento de um ímã em uma bobina e plota os sinais em tempo real.
